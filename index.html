Mech Programming Language

A simple, interpreted programming language designed for basic system operations, text processing, and automation tasks with an easy-to-learn syntax.

Get Started View Examples

Features

📝 Simple Syntax

Easy-to-read commands with minimal punctuation for quick learning and development.

🔊 Text-to-Speech

Built-in say command for audio output to make your programs more interactive.

🔄 System Control

Basic system operations like reboot and restart for automation tasks.

📂 File Importing

Modular code organization with the use command for better project structure.

🔢 Binary Conversion

Convert text to binary representation with the binary command.

⚖️ Conditional Logic

Basic if statements for flow control in your programs.

💻 Cross-Platform

Works on both Windows and Linux systems with consistent behavior.

🚀 Easy Installation

Simple setup process with minimal dependencies required.

Language Syntax

Basic Commands

# Print to console
print Hello, World!

# Text-to-speech
say Hello, I am a Mech

# Delay execution
sleep 2.5
zzz 1.0

# Binary conversion
binary Hello

# Conditional execution
if true: print This will execute

# Import other Mech files
use library.mech
use utilities/helpers.mech

# Comments
# This is a comment

Restart Command

⚠️ Important System Command

The restart command allows you to restart applications or services on your system. Use with caution as it will terminate and restart the specified process.

Usage

# Restart a specific application by name
restart "web server"

# Restart with a delay (in seconds)
restart database after 10

# Conditional restart based on a condition
if needs_restart: restart service

# Restart multiple services
restart nginx
sleep 5
restart mysql

Platform Support

🐧 Linux

Uses systemctl to restart services on Linux systems. Requires appropriate permissions.

🪟 Windows

Uses PowerShell commands to restart services on Windows systems.

Example: Service Monitor

# service_monitor.mech
# Monitor and restart a service if needed

print "Starting service monitor..."
say Service monitor activated

# Check if service is running
if not service_running("webapp"):
    print "Webapp service not running. Restarting..."
    restart webapp
    say Webapp service has been restarted
else:
    print "Webapp service is running normally"

Implementations

🐍 Mech (Python)

The original implementation of Mech built with Python 3.6+.

Installation

  1. Ensure you have Python 3.6+ installed
  2. Install the required dependency:
    pip install pyttsx3
  3. Download mech.py to your desired location

Usage

python mech.py filename.mech

⬢ WebMech (Node.js)

A Node.js port of Mech with the same syntax and functionality.

Installation

  1. Ensure you have Node.js installed
  2. You must download the seperate Webmech file that is in the code repository: WebMech (https://github.com/eotter-beep/mech/tree/main/webmech)

Usage

node webmech.js filename.mech
